# PathPal

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/junglemonkey112/Pathpal)

PathPal connects students with experienced college consultants to navigate US university admissions. Get personalised guidance on essays, school selection, standardised tests, and more.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

The easiest way to deploy PathPal is to use [Vercel](https://vercel.com):

1. Push this repository to GitHub (already done).
2. Go to [https://vercel.com/new](https://vercel.com/new) and import the repository.
3. Vercel auto-detects Next.js — no extra configuration is needed.
4. Click **Deploy**.

Alternatively, click the **Deploy with Vercel** button at the top of this README.

No environment variables are required — PathPal is fully client-side.

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router)
- [React 19](https://react.dev)
- [Tailwind CSS 4](https://tailwindcss.com)
